# Combine Kittens

Combine Kittens is an endless puzzle idle game where you merge cats to create bigger ones.
Developed in Unity for PC by two people in just one week, so expect a few bugs and surprises.

# Gameplay

Place kittens inside a box and combine identical ones to evolve them into bigger cats.
Keep merging to discover all the different types.

# Features

- Place cats in a box

- Merge two of the same cats to create a larger one

- When the box is full, you can watch an ad to clear space

- Ads are just funny cat videos, not real advertisements

- The score remains even after clearing the board

# Controls

+ A and D or Arrow Keys: Move cats left and right

+ Spacebar: Drop the cat into the box

# Tools Used

+ Unity 6

+ Procreate

+ Aseprite

# Known Issues

+ Sometimes the ads do not load correctly

+ There is no quit button, you must use Alt + F4 to exit

+ Some buttons may appear oddly sized but still work
